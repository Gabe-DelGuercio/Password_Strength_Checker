# Password_Strength_Checker
A simple password checker that anyone can create in order to get started on projects.

All it uses is some simple Python concepts in oder to find out if a given password is relitively weak or strong. 
Running the program is quite simple, as is can be either pasted into a terminal or into any website that will accept Python code.
All outcomes would be based off of the strength of the password you gave, depending on how many unique cases of letters, numbers, and symbols are used.
This, however, does not count for any nuanced passwords that are just generally more complex. For example, a random combination of twelve numbers is always going to be stronger than Abc123.
This is vewry simple, and some more intricate work is going to need to be done in order to create a password checker that looks out for more intricate strings of numbers and letters.
